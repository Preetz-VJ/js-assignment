Types of console

* To display message we use console.log() function
Eg: console.log("My first assignment");

* To display a part of of html document console.log() function
Eg: console.log(document.head);

* To display an error we use console.error() function
Eg: console.error("an error occurred");

* To display a warning message we use console.warn() function
Eg: console.warn("there is a warning");

* To display an objects we use console.log() function
Example: var a={name:"preet" , age: 21};
var b={name:"geet" , age: 23};
var c={name:"jeet" , age: 25};
Syntax: console.log({a,b,c});

* To display an objects in table format we use console.log() function
Example: var a={name:"preet" , age: 21};
var b={name:"geet" , age: 23};
var c={name:"jeet" , age: 25};
Syntax: console.table({a,b,c});

* To display an array we use console.log() function
Example: var language=[Hindi,English,Tamil,Kannada,Telugu];
var colour=['pink','blue','black','purple','green'];
Syntax: console.log([a,b]);

* To displaythe time taken to execute the script we use console.time() and console.timeEnd() function
Eg: console.time("time taken");
console.timeEnd("time taken");

* To display a customize message we use console.log() function
Eg: console.log("%c custom message","color:blue");
